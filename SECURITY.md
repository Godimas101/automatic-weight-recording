# 🔒 Security Policy

## 🚨 Do **not** share real secrets

This project works with private credentials. **Never** post real values in issues, pull requests, screenshots, workflow exports, or copied terminal output.

That includes:

- Wyze email / password
- `WYZE_KEY_ID`
- `WYZE_API_KEY`
- `WYZE_ACCESS_TOKEN`
- `WYZE_REFRESH_TOKEN`
- `GITHUB_PAT`
- server hostnames, SSH details, or `.env` contents

Use placeholders like `YOUR_API_KEY_HERE` in all public examples.

---

## 🛑 Before opening an issue or PR

Please double-check that you have removed or replaced:

- real tokens
- real account IDs
- personal email addresses
- private server names or IPs
- copied `.env` values

If a secret has already been exposed, **rotate it immediately** before doing anything else.

---

## 📬 Reporting a security issue

If you find a security problem in the workflow or docs, report it privately to the maintainer instead of posting exploit details publicly.

---

*Protect your automation like it owes you money.*
