# 🤝 Contributing to Automatic Weight Recording

Thanks for helping improve `automatic-weight-recording`.

---

## 🐞 Bugs and workflow issues

Please open a GitHub issue for:

- setup problems
- n8n workflow bugs
- Wyze token refresh issues
- README or install guide fixes
- enhancement ideas

When reporting a bug, **never include real credentials, tokens, or API keys**.

---

## 🔀 Pull requests

PRs are welcome.

Please:

1. keep changes narrowly scoped
2. use placeholder/example secrets only
3. scrub screenshots, logs, and workflow exports before posting
4. update the README when setup behavior changes

---

## 👀 Review flow

This repo uses code ownership so **`@Godimas101`** is automatically requested for review on pull requests.

---

*Automate responsibly. Commit carefully.*
